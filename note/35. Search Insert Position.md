# 35. Search Insert Position
### [题目链接](https://leetcode.com/problems/search-insert-position/description/)

超级简单，直接贴代码：

	class Solution {
	public:
	    int searchInsert(vector<int>& nums, int target) {
	        for(int i = 0; i < nums.size(); i++)
	        {
	            if(nums[i] >= target)
	                return i;
	        }
	        return nums.size();
	    }
	};
